# Web Agency WordPress Theme

A modern, high-conversion landing page theme built with **Bootstrap 5** and **ACF (Advanced Custom Fields)**. This theme is designed for digital agencies that want a modular, performant, and fully editable website.

## 🚀 Features
- **Fully Modular Architecture**: Built using `get_template_part()` for clean, sections-based development.
- **ACF Integration**: Every section is customizable via the WordPress admin (Hero, Services, Portfolio, etc.).
- **Performance Optimized**: Lightweight assets and Lucide icon integration.
- **Modern Design**: Premium slate dark and royal blue color palette with custom 'Outfit' and 'Inter' typography.

## 📁 Installation
1. Upload the `web-agency-theme` folder to your `/wp-content/themes/` directory.
2. In the WordPress Admin, go to **Appearance > Themes** and activate **Web Agency Theme**.
3. **Required Plugins**: Install and activate the **Advanced Custom Fields (PRO)** plugin to enable full editability.

## 🛠️ Content Setup
1. Create a new Page (e.g., "Home").
2. Under **Page Attributes**, set the template to the default or ensure it's set as your **Front Page** in **Settings > Reading**.
3. You will see several tabs in the ACF editor:
    - **Hero Section**: Edit heading, subtitle, and primary image.
    - **Client Logos**: Upload transparent SVG/PNG logos for the social proof section.
    - **Services**: A repeater field to manage service cards and Lucide icons.
    - **Benefits**: Manage your value propositions and outcomes.
    - **Portfolio**: Repeater for case studies with categories and links.
    - **Process**: Define your 1-2-3-4 step workflow.
    - **Contact**: Enter company info and optional form shortcodes.

## 🎨 Design System
- **Fonts**: Outfit (Headings), Inter (Body).
- **Primary Color**: `#2563EB` (Royal Blue).
- **Secondary Color**: `#0F172A` (Slate Dark).
